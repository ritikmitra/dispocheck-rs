# Security Policy

## Supported Versions

Only the latest published version of this crate receives security fixes.
Since it's a small, dependency-light, and frequently-republished crate,
please upgrade to the latest version rather than requesting backports.

## Reporting a Vulnerability

If you believe you've found a security vulnerability in this crate (as
opposed to a stale or incorrect domain entry — see [CONTRIBUTING.md](CONTRIBUTING.md)
for that), please report it privately using
[GitHub's private vulnerability reporting](../../security/advisories/new)
for this repository, rather than opening a public issue.

Please include:

- A description of the vulnerability and its potential impact.
- Steps to reproduce, or a proof-of-concept.
- The crate version affected.

We'll acknowledge reports within a few days and aim to publish a fix
(and, if applicable, a [RustSec advisory](https://rustsec.org/)) promptly
once a fix is available.
